# 3tier with terraform

해당 프로젝트는 `terraform`의 AWS 프로바이더를 사용하여 3tier을 구성하는 프로젝트입니다.

## 구성

### 프로젝트 파일 구성

- VPC
- Subnet
- Internet gateway
- Nat gateway
- Route Table
- Load Balancer
- EC2
